# Music Player Web Application

This is a simple music player web application developed using HTML, CSS, and JavaScript.
The project allows users to play, pause, and navigate between songs with a clean
and interactive user interface.

This project was created as **Task 4** of the **CodeAlpha Frontend Development Internship**.

---

## Project Description

The Music Player is designed to demonstrate the use of JavaScript for handling
audio playback and user interactions. It provides essential music player features
while maintaining a simple and responsive design.

---

## Features

- Play and pause audio
- Next and previous song navigation
- Display of song title
- Interactive progress bar
- Volume control
- Clean and modern UI design

---

## Technologies Used

- **HTML** – Structure of the music player
- **CSS** – Styling and layout
- **JavaScript** – Audio control and functionality

## How to Run the Project

1. Download or clone the repository
2. Make sure audio files are present in the project folder
3. Open `index.html` in any modern web browser
4. Use the controls to play and manage music
